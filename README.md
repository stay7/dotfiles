## install using brew
- zsh
- fzf
- starship

## zim module install
`curl -fsSL https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh`

`zimfw install`

## install starship
`curl -sS https://starship.rs/install.sh | sh`

## asdf plugin
`asdf plugin add rust`
`asdf plugin add starship`

`asdf list all rust`

`asdf install rust <version>`
